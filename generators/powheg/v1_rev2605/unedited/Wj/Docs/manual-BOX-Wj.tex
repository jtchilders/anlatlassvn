\abstract{This note documents the use of the package
  \POWHEGBOX{} for $W^\pm+1$~jet production processes.
  Results can be easily interfaced to shower Monte Carlo programs, in
  such a way that both NLO and shower accuracy are maintained.}

\section{Introduction}

The \POWHEGBOX{} program is a framework for implementing NLO
calculations in Shower Monte Carlo programs according to the
\POWHEG{} method. An explanation of the method and a discussion of
how the code is organized can be found in
refs.~\cite{Nason:2004rx,Frixione:2007vw,Alioli:2010xd}.  The code is
distributed according to the ``MCNET GUIDELINES for Event Generator
Authors and Users'' and can be found at the web page 
%
\begin{center}
 \url{http://powhegbox.mib.infn.it}.
\end{center}
%
In this manual, we describe the \POWHEG{} NLO implementation of $W^{\pm}+1$~jet
hadroproduction, with $W^{\pm} \to \ell^{\pm} \bar{\nu}_\ell$ fully taken into account at
leading order, closely following the implementation in  ref.~\cite{Alioli:2010qp}


\section{Generation of events}

Build the executable\\
\tmtexttt{\$ cd POWHEG-BOX/Wj \\
\$ make pwhg\_main }\\

\noindent
Then do (for example) \\
\tmtexttt{\$
cd testrun-lhc\\
\$ echo wp | ../pwhg\_main}\\

\noindent
At the end of the run, the file \tmtexttt{wp-events.lhe} will contain 50000
events for $W^+ + 1$~jet hadroproduction in the Les Houches format.


\noindent
In order to shower them with \PYTHIA{} do\\
\tmtexttt{\$
cd POWHEG-BOX/Wj \\ 
\$ make main-PYTHIA-lhef \\ 
\$ cd testrun-lhc \\
\$ echo wp | ../main-PYTHIA-lhef}

\section{Process specific input parameters}

The decay of the $W$ boson is controlled by the token \tmtexttt{vdecaymode},
in this way:

\begin{itemize}
\item[-] 1 for $e$ channel
\item[-] 2 for $\mu$ channel
\item[-] 3 for $\tau$ channel
\end{itemize}
whose masses ({\tt physpar\_ml}), together with all other input physical
parameters, are set in the \tmtexttt{init\_couplings.f} file.
